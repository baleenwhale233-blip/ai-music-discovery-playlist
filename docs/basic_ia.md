# 基础信息架构 v0.2

## 1. 信息架构原则

新版本信息架构围绕一条主线：

```text
发现目录
-> 筛选条目
-> 保存为本地音频
-> 加入听单
-> 原生播放器连续收听
```

核心原则：

- 广场是目录，不是公开播放器
- 播放发生在用户主动保存到本地之后
- B 站是大陆版默认公开来源
- YouTube 是技术支持但大陆默认隐藏的来源
- 学习播放单允许成立，但不抢首页主心智

## 2. 一级导航

### 2.1 发现

目标：帮助用户找到值得保存为本地听单的目录。

内容模块：

- 编辑精选目录
- AI 音乐专题
- 最新收录
- 热门目录
- 学习播放单入口

首期约束：

- 不直接播放
- 不自动缓存
- 点击目录进入目录详情

### 2.2 听单

目标：管理用户自己的本地听单。

内容模块：

- 我的听单
- 最近缓存
- 最近播放
- 缓存中
- 缓存失败

听单内能力：

- 播放
- 暂停
- 继续
- 拖动进度
- 下一首
- 移除条目
- 删除本地缓存

### 2.3 导入

目标：让用户从来源链接创建目录或听单。

导入方式：

- 单条 B 站链接
- B 站收藏夹链接
- B 站播放列表链接
- YouTube 链接（隐藏/实验）

导入后状态：

- 候选目录
- 用户筛选
- 保存为本地音频
- 加入听单

### 2.4 我的

目标：管理设置、缓存、实验功能和来源能力。

内容模块：

- 存储空间
- 清理缓存
- 实验功能
- 来源开关
- 账号绑定
- 关于

## 3. 核心页面

### 3.1 目录详情页

用途：展示一个来源目录里的所有候选内容。

内容：

- 目录标题
- 来源平台
- 创建者/UP 主
- 条目列表
- 每条标题、封面、时长、来源
- 是否已缓存
- 是否已加入听单

操作：

- 删除不相关条目
- 单条保存为本地音频
- 批量保存选中内容
- 加入已有听单
- 新建听单

### 3.2 候选筛选页

用途：收藏夹或播放列表导入后，让用户先清理不想听的内容。

内容：

- 候选条目
- 默认全选
- 支持取消选择
- 支持删除条目
- 支持按标题搜索
- 后续可加 AI 粗筛

操作：

- 保存选中内容
- 取消导入
- 只保存当前条

### 3.3 缓存进度页

用途：展示本地音频生成进度。

内容：

- 当前正在缓存的条目
- 成功数量
- 失败数量
- 失败原因
- 队列状态

操作：

- 暂停缓存
- 继续缓存
- 跳过失败
- 重试失败

MVP 可先不单独做完整页面，用听单详情里的状态展示代替。

### 3.4 播放器页

用途：本地音频播放。

内容：

- 封面
- 标题
- 来源平台
- 作者/UP 主
- 当前听单
- 进度条
- 当前时间/总时长

操作：

- 播放
- 暂停
- 继续
- 拖动进度
- 上一首
- 下一首
- 删除缓存
- 打开来源链接

### 3.5 听单详情页

用途：管理一个本地听单。

内容：

- 听单标题
- 条目数量
- 已缓存数量
- 未缓存数量
- 条目列表

操作：

- 播放全部
- 缓存全部未缓存
- 删除条目
- 调整顺序
- 清理本地缓存

## 4. 关键状态

### 4.1 目录项状态

- 未缓存
- 缓存中
- 已缓存
- 缓存失败
- 已删除

### 4.2 播放状态

- idle
- loading
- playing
- paused
- ended
- error

### 4.3 来源状态

- 公开可见
- 隐藏实验
- 地区可见
- 暂不可用

## 5. 来源露出策略

### 大陆版默认

- B 站：公开
- YouTube：隐藏
- TikTok：隐藏
- 抖音：实验

### 国际版默认

- YouTube：公开
- B 站：可选
- TikTok：实验

### 隐藏触发

建议：

- 设置页连续点击版本号 7 次
- 打开“实验功能”
- 显示隐藏来源开关

隐藏来源不进首页，不做默认推荐。

## 6. 广场结构

广场展示的是目录，不是音频文件。

推荐模块：

- 今日 AI 音乐目录
- 编辑精选听单目录
- 最新收录目录
- 热门收藏目录
- 学习播放单目录

点击后的路径：

```text
广场目录卡片
-> 目录详情
-> 选择条目
-> 保存为本地音频
-> 加入听单
```

## 7. 导入流程

### 单条链接

```text
粘贴链接
-> 解析元信息
-> 预览
-> 保存为本地音频
-> 加入听单
-> 播放
```

### 收藏夹/播放列表

```text
粘贴目录链接
-> 解析候选条目
-> 用户删除不相关内容
-> 批量保存选中条目
-> 生成听单
-> 连续播放
```

## 8. 空状态

### 发现页空状态

- 展示编辑精选目录
- 引导用户导入 B 站链接

### 听单空状态

- “还没有本地听单”
- 主按钮：“导入 B 站链接”
- 次按钮：“从收藏夹创建”

### 缓存失败状态

- 显示失败原因
- 支持重试
- 支持打开来源链接
- 支持删除该条目

## 9. 不进首页的能力

这些能力可以存在，但不作为首页主入口：

- 学习播放单
- YouTube 实验来源
- TikTok 实验来源
- 抖音实验来源
- 创作者认领
- AI 元信息编辑

## 10. MVP 页面优先级

### P0

- 导入页
- 单条链接预览
- 本地缓存状态
- 听单详情
- 播放器页

### P1

- 收藏夹候选筛选页
- 批量缓存状态
- 最近缓存
- 缓存失败列表

### P2

- 发现页
- 编辑精选目录
- 学习播放单目录

### P3

- 实验来源设置
- 账号绑定
- 创作者页
