# Web Information Architecture

## 导航原则

Mobile Web Alpha 的导航围绕听单创建、目录管理和本地播放展开。添加来源入口收敛在创建或编辑听单流程里，避免把导入动作拆成独立主 tab。

一级体验优先级：

- 听单列表和听单详情。
- 创建听单与从链接添加。
- 已缓存音频播放器。
- 个人缓存和账号管理。

## 顶层入口

- `/login`: Alpha 登录。
- `/playlists`: 听单列表和目录入口。
- `/playlists/new`: 创建或编辑听单草稿。
- `/playlists/new/add`: 从来源链接添加候选条目。
- `/playlists/[playlistId]`: 听单详情、条目筛选、缓存操作、收藏入口。
- `/playlists/[playlistId]/manage`: owner 管理听单标题、简介和条目。
- `/player`: 全局完整播放器。
- `/me`: 我的草稿、已发布目录、本地缓存和账号操作。

根路径 `/` 可以指向 `/playlists` 的同一产品入口。

## 页面地图

```text
/login
/
+-- /playlists
|   +-- /playlists/new
|   |   +-- /playlists/new/add
|   +-- /playlists/[playlistId]
|       +-- /playlists/[playlistId]/manage
|       +-- /player
+-- /me
    +-- /playlists/new
    +-- /player
```

## 核心流程

创建听单：

```text
/playlists
-> /playlists/new
-> /playlists/new/add
-> 解析来源链接
-> 选择候选条目
-> /playlists/new
-> 发布听单
-> /playlists/[playlistId]
```

缓存与播放：

```text
/playlists/[playlistId]
-> 选择可缓存条目
-> 用户确认缓存
-> 缓存完成
-> mini player
-> /player
```

个人管理：

```text
/me
-> 查看草稿、已发布听单、本地缓存
-> 打开草稿或播放器
-> 退出登录
```

## 页面职责

### `/login`

- 输入 Alpha 登录信息。
- 保存访问令牌。
- 登录成功后进入听单列表。
- 令牌失效时作为重新登录入口。

### `/playlists`

- 展示已发布听单目录。
- 展示示例目录和用户本地发布目录。
- 提供创建听单入口。

### `/playlists/new`

- 管理当前听单草稿。
- 展示已添加候选条目。
- 支持调整顺序、删除条目和发布听单。
- 提供从链接添加入口。

### `/playlists/new/add`

- 输入 B 站等来源链接。
- 展示当前可用来源。
- 解析单条、收藏夹、播放列表或合集链接。
- 展示候选结果并支持选择条目加入草稿。

### `/playlists/[playlistId]`

- 展示听单封面、作者、条目数量和缓存数量。
- 展示目录条目。
- 支持选择可缓存条目。
- 触发用户主动缓存。
- 支持收藏或取消收藏。
- owner 可进入管理页。
- 提供打开播放器入口。

### `/playlists/[playlistId]/manage`

- 仅 owner 可管理。
- 支持编辑标题和简介。
- 支持删除条目。
- 保留添加来源和排序能力的扩展空间。

### `/player`

- 展示当前可播放的本地缓存队列。
- 使用真实 media element 播放音频。
- 支持播放、暂停、继续、拖动进度和播放结束后进入下一首。
- 支持上一首、下一首、顺序播放、列表循环、单曲循环和随机播放。
- 队列和底部 mini player 使用同一个播放器状态。

### `/me`

- 展示草稿数量、已发布数量和已缓存数量。
- 展示用户收藏的听单目录。
- 展示用户创建的听单目录。
- 展示最近本地缓存。
- 提供退出登录。

## 空状态与错误状态

- 未登录：显示登录提示并引导到 `/login`。
- 无草稿：提示先从链接添加条目。
- 无解析结果：提示检查来源链接。
- 候选为空：允许返回链接输入页重新解析。
- 无可缓存条目：禁用缓存按钮并展示当前状态。
- 缓存失败：在条目状态中展示失败状态，并保留重试空间。
- 播放器为空：提示先在听单详情缓存条目。
- 令牌失效：清理本地令牌并提示重新登录。

## 后续扩展

- 听单详情内的批量重试和缓存进度展示。
- 听单编辑入口。
- 资料库或历史导入入口。
- 更完整的本地缓存空间管理。
- 来源能力灰度开关。
- 桌面 Web 的密度优化。
